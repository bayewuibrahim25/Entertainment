# Ibrahim Entertainment Picks

## Website Overview
Ibrahim Entertainment Picks is a personal portfolio website created as part of the Web Authoring module. The website showcases my favourite entertainment choices, including music, movies, and games, while demonstrating my skills in HTML5 and CSS3.

## Page Descriptions

### index.html
The homepage introduces the website and highlights my top entertainment picks. It includes images, text content, and a data table displaying ratings.

### gallerryreviews.html
This page provides more detailed entertainment content, including lists of favourite artists, movies, and games. It demonstrates the use of ordered and unordered lists and structured sections.

### contact.html
The contact page includes a form where users can enter their name, email address, and a message. This page demonstrates form elements and accessibility features such as labels.

## Special Features
- Multi-page navigation
- Responsive layout
- Data table for entertainment ratings
- Images with descriptive alt text
- Contact form with multiple input types

## Technologies Used
- HTML5
- CSS3
- GitHub Pages

## How to View the Site Locally
1. Download or clone the GitHub repository.
2. Open index.html in any modern web browser.
3. Use the navigation menu to view all pages.

The website is hosted using GitHub Pages and can be viewed online.
